John's Rinky Dink Calculator Manual

You clicked the "?" button or "About" button which took you here. John's Rinky Dink Calculator is a simple basic digital calculator made by a first-year college student named John for his midterm project. If you wish to go back to the calculator only, press the "Home" button then the "Go to Calculator" button.

It works like a normal calculator. Press the first value, select an operation, enter another value, then press "=" for results. If you want to delete a single value, there is a button with an arrow pointed to the left. That is the backspace button. Pressing it will delete one digit to the text box. If you want to start over, Press the AC (All Clear) button to wipe the whole operation.

If "Error" or "NaN" appears, it means the input is invalid or mathematically undefined. If undefined appeared, it means you didn't put anything and just pressed the "=" button. If Function Error appeared with some symbols, then you got the same type of error twice in a row. Press AC before adding another digit to avoid further errors.
